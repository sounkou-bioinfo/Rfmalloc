# Number of variants in a banded LD store

Number of variants in a banded LD store

## Usage

``` r
ld_ncol(store)
```

## Arguments

- store:

  An
  [fmalloc_ld](https://sounkou-bioinfo.github.io/Rfmalloc/Rfmalloc/reference/fmalloc_ld.md)
  object.

## Value

The number of variants (columns == rows) as an integer.
